Overview

Tree Style Tab Manager ("the Extension") is committed to protecting your privacy. This privacy policy explains how the Extension handles your data.

Data Collection

The Extension does NOT collect, store, transmit, or share any personal data.

What Data is Processed Locally

The Extension processes the following data locally on your device only:

  1. Tab Information
    • Tab titles
    • Tab URLs
    • Tab favicons
    • Tab relationships (parent-child hierarchy)
    • Tab state (active, pinned, etc.)
  2. User Settings
    • Tab restoration window preference
    • UI preferences
  3. Virtual Tabs
    • Information about recently closed tabs
    • Last accessed timestamps
    • Expiration times

How Data is Stored

All data is stored locally using Chrome's built-in storage API:

  • chrome.storage.local - Stores tab tree structure and settings on your device
  • Data never leaves your computer
  • Data is not synchronized across devices
  • Data is deleted when you uninstall the extension

Permissions Explained

The Extension requests the following permissions:

Required Permissions

  1. tabs - Read tab information (title, URL, favicon)
    • Used to display tabs in the tree structure
    • Required for basic functionality
  2. storage - Store data locally
    • Saves your tab tree structure
    • Saves your preferences
    • All data stays on your device
  3. activeTab - Activate tabs
    • Required to switch between tabs when clicked
    • No additional data access
  4. tabGroups - Manage tab groups
    • Organize tabs in groups
    • Future feature support
  5. sidePanel - Display in Chrome's side panel
    • Native Chrome integration
    • Better user experience
  6. system.memory - Monitor memory usage
    • Optimize cleanup operations
    • Prevent memory leaks
    • No personal data collected

Content Scripts

The Extension injects content scripts on all pages to:

  • Detect when pages are loaded (for tree organization)
  • Detect navigation events

Does NOT: Read page content, track browsing, or collect personal information

Data Sharing

The Extension does NOT share data with anyone.

  • ✗ No data is sent to external servers
  • ✗ No analytics or tracking
  • ✗ No third-party services
  • ✗ No ads or monetization
  • ✗ No user profiling

Data Security

  • All data is stored locally on your device using Chrome's secure storage API
  • No transmission of data over the internet
  • No backend servers or databases
  • No cloud storage

Your Rights

Since all data is stored locally on your device:

  • Access: You can view all stored data through the Extension
  • Delete: Uninstall the Extension to delete all data
  • Control: You have full control over your data at all times

How to Delete Your Data

  1. Clear tab tree: Click "Clear Tree Structure" button in the side panel
  2. Clear virtual tabs: Click "Clear Virtual Tabs" button
  3. Remove extension: Uninstall the Extension to delete all stored data

Third-Party Services

The Extension does NOT use any third-party services:

  • No analytics (Google Analytics, etc.)
  • No crash reporting
  • No advertising networks
  • No social media integration

Children's Privacy

The Extension does not knowingly collect information from children under 13 years of age. The Extension is not directed at children.
